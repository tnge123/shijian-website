# 北京实见信息技术有限公司官网

## 项目介绍

服务国央企的人力资源整合服务商官网，采用 Next.js + Tailwind CSS 构建。

## 技术栈

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Lucide React (图标)

## 功能特点

- 响应式设计，完美适配各种设备
- 现代化 UI，红白专业配色
- 滚动导航栏效果
- 自动轮播的业务架构展示
- 平滑滚动锚点导航

## 页面结构

1. **导航栏** - 固定顶部，滚动时变化样式
2. **Hero区域** - 全屏banner，AI访谈主题
3. **产品卡片** - 5个核心产品展示
4. **业务架构** - Tab切换展示，自动3秒轮播
5. **客户案例** - 客户LOGO墙 + 标杆案例
6. **管理洞察** - 文章卡片列表
7. **Footer** - 完整的网站底部信息

## 本地开发

```bash
# 安装依赖
npm install

# 开发服务器
npm run dev

# 构建
npm run build
```

## 部署

项目配置为静态导出，构建后文件位于 `dist` 目录。

```bash
npm run build
```

## 配色方案

- 主色：#DC2626 (红色)
- 背景：#FFFFFF (白色) / #F5F5F5 (浅灰)
- 文字：#171717 (深灰) / #737373 (中灰)
